# railtracks-shipyard

Python package scaffold with a primary `game-engine` entry point.

## Project structure

- `src/railtracks_shipyard/` - package root
- `src/railtracks_shipyard/game_engine/` - game engine scaffold module
- `tests/` - focused scaffold tests

## Quick start

```bash
python -m pip install -e .
python -m unittest discover -s tests -v
```
